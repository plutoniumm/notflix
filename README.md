### Keyboard Shortcuts
Other than the basic youtube shortcuts

| Key | Action |
| --- | --- |
| `l` | Open Listing |
| `p` | picture-in-picture |
| `b` | brightness toggle |
| `n` | next |
|  `m` | mute |
| `c` | toggle subtitles |
| `f` | toggle fullscreen |
| digit | seek to that*10 percent of vid |
